Architecture

Storage: New player_achievements table — player_name, achievement_id, unlocked_at

Detection: src/lib/achievements.ts — checkAchievements(gameState, playerName) called at round-end and game-end. Pure function, returns newly unlocked IDs.

Display: Gold toast (uses existing GameToast with 'celebration' style). Badge grid on PlayerProfileModal. Achievement list on a new tab in StatsLeaderboard.

16 achievements to start — enough for discovery without grind. Easy to add more later.
