Technical Approach

Pure CSS — no animation library. Uses perspective, transform-style: preserve-3d, backface-visibility: hidden on Card.tsx.

New file: src/index.css additions — keyframes for deal-arc, draw-slide, discard-toss, meld-slam, shuffle-riffle.

Modified: Card.tsx (3D flip wrapper), HandDisplay.tsx (deal animation), GameBoard.tsx (draw/discard transitions), BuyingCinematic.tsx (enhanced snatch).

Performance: All animations use transform and opacity only — composited by GPU, no layout thrashing. will-change: transform on animated cards.
